⚠️ Important: Changing the installation path requires manual updates to your web server configuration. Make sure to update your Apache/Nginx settings accordingly.

How to Change Installation Path

  1. Move all files to your new directory
  2. Update your web server configuration (Apache .htaccess or Nginx config)
  3. Edit config.php and update these values:
    'app_url' => 'https://yourdomain.com/new-path',
    'base_path' => '/new-path',
    'install_path' => '/full/server/path/to/new-path',
  4. Clear your browser cache
  5. Test the application at the new location
