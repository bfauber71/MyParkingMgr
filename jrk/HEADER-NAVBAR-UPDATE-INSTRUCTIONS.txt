TECHNICAL NOTES
================================================================================

HTML Structure:
- <header class="app-header" id="appHeader"> - Brand, logo, clock, license badge
- <nav class="navbar" id="navbar"> - Navigation dropdown menu
- Both hidden by default (style="display: none")
- Both shown when user logs in via JavaScript

CSS Architecture:
- .app-header: Fixed position, top: 0, z-index: 1001
- .navbar: Fixed position, top: 56px (mobile) or 72px (desktop), z-index: 1000
- .container: margin-top: 112px (mobile) or 140px (desktop)
- .settings-tabs: sticky, top: 112px (mobile) or 140px (desktop)
- .nav-dropdown-menu: fixed, top: 104px (mobile)

JavaScript Changes:
- showLogin(): hides appHeader + navbar
- showDashboard(): shows appHeader + navbar (display: flex)
- Clock updates: still uses #navDate and #navTime IDs (in header now)
- License badge: still uses #licenseStatusBadge ID (in header now)

Removed Styles:
- .nav-brand, .brand-content, .brand-name, .brand-version
- .brand-logo, .nav-clock, .nav-date, .nav-time
- These were replaced with header-specific equivalents
