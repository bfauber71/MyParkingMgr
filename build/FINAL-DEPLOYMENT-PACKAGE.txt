## Deployment Instructions

1. **Upload Files:**
   - Extract ZIP to your server via FTP/SFTP
   - Ensure .htaccess is uploaded (hidden file)

2. **Configure Database:**
   - Edit config.php with MySQL credentials
   - Run setup.php in browser

3. **SSL Certificate:**
   - Install SSL certificate (required for HTTPS)
   - Most hosts offer free Let's Encrypt certificates

4. **Test:**
   - Login: admin / admin
   - Navigate to Settings > Violations
   - Verify all 10 violation types display
   - Test trial badge displays
